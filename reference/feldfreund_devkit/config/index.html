    GnssConfiguration dataclass ¶

    GnssConfiguration(
    *,
    x: float = 0.041,
    y: float = -0.255,
    z: float = 0.6225,
    yaw: float = 0.0
)

    Configuration for the GNSS of the Feldfreund robot.

    X, Y, Z are the position of the main GNSS antenna. The yaw is the direction to the auxiliary antenna. It should be 90, but the offset is configured in the Septentrio software.

    Defaults

    x: 0.041 y: -0.255 z: 0.6225 yaw: 0.0
